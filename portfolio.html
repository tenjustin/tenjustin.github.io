StockManager

Ce projet est une application client lourd de gestions des stocks pour une petite épicerie. Il fallait que l'application puisse tourner en local sur l'ordinateur disponible dans la pièce dédié au stock de l'épicerie. Pour ce faire j'ai décidé de développer une application simple en C# avec une base de données locale Microsoft SQL Server.
